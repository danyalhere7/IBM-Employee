#  IBM Employee Attrition Prediction

Predicting employee attrition based on HR analytics data from IBM.

##  Dataset
- Target: `Attrition`
- Features: Age, JobRole, MonthlyIncome, etc.

##  Libraries
- Pandas, NumPy, Seaborn, Scikit-learn
- SMOTE for class balance

##  Evaluation
- Accuracy, ROC AUC, Confusion Matrix
